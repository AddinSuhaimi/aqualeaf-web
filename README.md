# AquaLeaf Web Application

AquaLeaf is a smart seaweed farming management system designed to assist farmers and administrators in monitoring seaweed quality, managing reports, and analyzing farm data efficiently.

---

## Live Application

The AquaLeaf web application is deployed and accessible online:

https://aqualeaf-web.vercel.app/

## Testing Guidelines

* Use the **live deployed system** for functional testing
* Data may be reset periodically

---

## Running Locally

### 🖥️ Prerequisites

Ensure the following are installed:

* Node.js (v18 or above)
* npm (comes with Node.js)
* Git
* Visual Studio Code (recommended)

### 📦 Installation

Clone the repository:

```bash
git clone https://github.com/AddinSuhaimi/aqualeaf_app.git
cd aqualeaf_app
```

Install dependencies:

```bash
npm install
```

Run the Application:

```bash
npm run dev
```

Then open:

```text
http://localhost:3000
```
